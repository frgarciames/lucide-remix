# Lucide Remix

Implementation of the Lucide icon library for Remix 3 applications.

## Installation

```sh
pnpm add lucide-remix
```

```sh
npm install lucide-remix
```

```sh
yarn add lucide-remix
```

```sh
bun add lucide-remix
```

## Usage

```ts
import { Camera } from 'lucide-remix'
```

Or import individual icons for optimal tree-shaking:

```ts
import { Camera } from 'lucide-remix/icons/camera'
```
